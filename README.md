# 城市资产负债表

这是一个用于记录在不同城市生活体验的网页应用。你可以添加城市，并为每个城市添加正面和负面的生活体验。

## 功能特点

- 添加新的城市
- 为每个城市添加正面和负面的生活体验
- 使用不同颜色区分正面和负面体验
- 响应式设计，支持各种设备

## 安装和运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm start
```

3. 在浏览器中打开 http://localhost:3000 查看应用

## 技术栈

- React
- TypeScript
- Tailwind CSS
